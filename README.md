# ML-Based Virtual Career Counselling

An intelligent web-based career guidance platform that leverages **Machine Learning** to recommend suitable career paths based on a user's interests, skills, and personality traits. The application provides personalized career recommendations, learning resources, and career roadmaps through an interactive web interface.

## 🚀 Features

- User Registration & Authentication
- Basic Career Assessment
- Advanced Domain-Specific Assessment
- Machine Learning-Based Career Prediction
- Personalized Career Recommendations
- Skill Gap Analysis
- Learning Resources & Course Recommendations
- Career Roadmap Generation
- Assessment History
- Responsive User Interface

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/ML-Based-Virtual-Career-Counselling.git
```

### 2. Navigate to the project directory

```bash
cd ML-Based-Virtual-Career-Counselling
```

### 3. Create a virtual environment

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux/Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Apply migrations

```bash
python manage.py migrate
```

### 6. Run the development server

```bash
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```

## 🧠 How It Works

1. User registers or logs into the application.
2. Completes the career assessment questionnaire.
3. Responses are processed by the Machine Learning model.
4. The model predicts the most suitable career domain.
5. The application displays:
   - Career Prediction
   - Confidence Score
   - Recommended Skills
   - Learning Resources
   - Suggested Career Roadmap
